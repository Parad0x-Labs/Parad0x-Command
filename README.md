Local-first desktop command center. Live graph of apps, processes, browser activity, files, agents, and system telemetry. No cloud dependency.

## What you get

- Live system graph: apps, processes, files, hardware, browser activity.
- `SYSTEM` view for operational control. `AI MAP` view for live agent activity.
- Local process controls and network cut/restore.

## Quick start

```bash
# macOS / Linux
git clone https://github.com/Parad0x-Labs/Parad0x-Command.git && cd Parad0x-Command && bash ./setup.sh
```

```powershell
# Windows
git clone https://github.com/Parad0x-Labs/Parad0x-Command.git; Set-Location .\Parad0x-Command; .\setup.ps1
```
